# dotfiles

My personal configuration files. Built on [Dotbot](https://github.com/anishathalye/dotbot) for cross-platform compatibility.

## Installation

For linux and macOS:
```bash
./install
```

For windows:
```powershell
.\install.ps1
```

## Updating

### Dotbot + Plugins

```bash
git submodule update --remote
```

## Completions
```bash
zprezto-build-completion
```
